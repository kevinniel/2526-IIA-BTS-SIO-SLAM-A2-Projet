# Analyse de l'existant

## Qu'est-ce que c'est ?

- C'est un document qui décrit les logiciels, systèmes et infrastructures déjà en place.
- Peut intégrer les compétences techniques

## A quoi ça sert ?

- Identifier les composants réutilisables ou obsolètes
- Servir de base pour définir les choix techniques futurs

## Ce qu'on peut retrouver dedans

- Description des systèmes existants
- Forces & Faiblesses de l'existant
- Recommandation d'évolutions
