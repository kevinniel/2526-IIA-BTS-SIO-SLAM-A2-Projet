# Documentation utilisateurs

## Qu'est-ce que c'est ?

- Ensemble de documents destinés aux utilisateurs finaux de la solution.
- Ils expliquent l'utilisation fonctionnelle, sans entrer dans le détail technique.

## A quoi ça sert ?

- Faciliter la prise en main de l'outil par les utilisateurs.
- Réduire les erreurs d'utilisation et les demandes de support.
- Garantir un usage conforme aux processus métier.

## Ce qu'on peut retrouver dedans

- Un guide de démarrage et les prérequis d'utilisation.
- Les procédures fonctionnelles pas à pas.
- Les cas fréquents d'erreur et leur résolution.
- Une FAQ et les contacts de support.
